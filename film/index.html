In Development

Sanda Seivom
Cinematographer / Color Design · Pre-production

A combat-sport drama set in Kannagi Nagar, Chennai — where a community's game becomes its weapon against a corrupted power. Script at draft 3 with scene breakdown. Below: how the film's color language is being designed, from the streets of the location to the final base palette and custom LUTs.

01Location Survey

The palette starts where the story lives. Kannagi Nagar photographed and street-viewed block by block — its resettlement housing, festival nights, and sun-bleached facades set the film's chromatic ground truth.

02Palette Extraction

Colors sampled directly from the location — steel-blue sky, lime-washed walls, dust, rust and autorickshaw orange — then structured on the wheel. Chosen harmony: triad.

03Paint Tests

The triadic scheme tested physically — watercolor mixes on paper before a single frame is graded. If it doesn't hold as pigment, it won't hold on screen.

04Texture & Graffiti

Faded-tone graffiti and weathered-wall references define how color should age in the frame — worn, layered, sun-eaten. Watercolor textures guide diffusion and bleed for the grade.

05Base Palette

Everything resolves into the working base palette — the master swatch set that drives production design, costume, and the custom LUTs built for on-set monitoring and the final grade.
