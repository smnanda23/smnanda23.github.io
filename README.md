# Nandha – AI, Computer Vision & VFX Portfolio

This repository hosts my personal website at **smnanda23.github.io**, showcasing my work across AI engineering, computer vision, generative media, and VFX. The site is built as a lightweight, fast-loading HTML portfolio optimized for visual projects and technical case studies.

## About Me

I am an AI and Computer Vision engineer with experience across:

- Generative AI models for image and video.
- Computer vision pipelines for segmentation, tracking, and enhancement.
- VFX workflows that bridge traditional post-production with modern AI.
- Cloud GPU infrastructure and deployment of ML workloads.

I focus on building practical, production-ready systems—ranging from experimentation notebooks to fully deployed pipelines on cloud platforms.

## What’s Inside This Repo

The repository currently contains:

- `index.html` – Main landing page for the portfolio site.
- `assets/` – Static assets (images, styles, scripts) used across the site.
- `film/` – Film, photography, and visual project pages, including stills, breakdowns, and behind-the-scenes notes.

The site is served via **GitHub Pages**, with automated deployments tracked under the repository’s Deployments section. [page:1]

## How to Run Locally

You can view the site locally with any static file server:

```bash
git clone https://github.com/smnanda23/smnanda23.github.io.git
cd smnanda23.github.io
# Option 1: Simple Python HTTP server
python -m http.server 8080

# Then open http://localhost:8080 in your browser.
```

No build step is required; the site runs as plain static HTML.

## Deployment

This site is deployed using **GitHub Pages**:

- Branch: `main`
- Hosting: `https://smnanda23.github.io`
- Deployments: Managed automatically via GitHub Pages (see “Deployments” in the repo). [page:1]

Any commits pushed to `main` are picked up by GitHub Pages and deployed automatically within a short time window.
